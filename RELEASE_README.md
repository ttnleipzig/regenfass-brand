# Regenfass Corporate Identity Assets - v${VERSION}

## 📦 What's Included

This release contains all official Corporate Identity and Corporate Design materials for Regenfass.

### Assets Structure

- **Logos**: All logo variants in SVG and PNG formats
  - Dark background versions (purple and red)
  - Light background versions (red)
  - Favicon files in multiple formats
  - Located in `assets/logos/`

- **Colors**: Brand color palette definitions
  - `colors.css` - CSS variables (in `assets/colors/`)
  - `colors.json` - JSON format with HEX, RGB, and CMYK values (in `assets/colors/`)
  - Color swatches in SVG format (in `assets/colors/swatches/`)

- **Fonts**: Typography specifications and font files (if included)
  - Located in `assets/fonts/`

- **Templates**: Document and presentation templates
  - Located in `assets/templates/`

- **Guidelines**: Complete brand guidelines documentation
  - Brand Guidelines
  - Logo Usage
  - Color Palette
  - Typography
  - Located in `assets/guidelines/`

## 🚀 Quick Start

1. **Download the ZIP file**
2. **Extract** the ZIP file to your project directory
3. **Review the guidelines** in the `assets/guidelines/` directory before using assets
4. **Follow brand guidelines** to ensure consistent brand representation

## 📖 Usage Guidelines

### For Developers

```bash
# Extract the assets
unzip regenfass-assets-v${VERSION}.zip

# Use CSS variables for colors
@import './assets/colors/colors.css';

# Or use JSON for JavaScript/TypeScript projects
import colors from './assets/colors/colors.json';
```

### For Designers

- Use vector formats (SVG) for scalable graphics
- Use PNG for transparent backgrounds
- Refer to `assets/guidelines/` for proper usage rules
- Maintain minimum clear space around logos

### For Marketing

- Use templates from `assets/templates/` directory
- Use logos from `assets/logos/` directory
- Follow brand guidelines in `assets/guidelines/` for all materials
- Use appropriate file formats for your medium (print vs. digital)
